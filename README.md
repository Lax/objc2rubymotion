Ojbc2Rubymotion:
===============
**Objective-C to Rubymotion Converter**

Based on the [Sublime-ObjcToRubyMotion](https://github.com/thinkclay/Sublime-ObjcToRubyMotion) plugin.

Implementation
--------------

Coffeescript mostly; on Sinatra.

Install it:

`bundle install`

Run it:

`shotgun -p 3000`

Visit it:

`localhost:3000`

Test it:

`localhost:3000/test`
